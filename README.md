# Braids & Beauty - Premium Braiding Salon in Torino

A modern, responsive, and luxurious website for Braids & Beauty, a specialized African braiding salon offering home service in Torino, Italy.

## ✨ Premium Features

- **🎨 Luxe Design**: Warm brown dominant, rose secondary, white, and gold touches with hexagonal decorative elements
- **🌙 Dark Mode**: Girly premium dark mode with smooth transitions and localStorage persistence
- **📱 Fully Responsive**: Mobile-first design optimized for all devices
- **🎬 Premium Animations**: Scroll-triggered fade-in effects, parallax hero, staggered gallery animations
- **💬 WhatsApp Integration**: Floating button and form-integrated booking with pre-filled messages
- **📊 Scroll Progress**: Elegant progress bar at the top of the page
- **🎯 Micro-interactions**: Hover glow effects, button ripples, form focus animations
- **🔍 SEO Optimized**: Local SEO with proper meta tags, semantic HTML, and accessibility features
- **⚡ Performance**: Optimized animations and smooth transitions
